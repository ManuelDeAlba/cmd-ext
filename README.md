# cmd-ext
#### Custom cmd commands

## Quick start
Download the code and install the dependencies

> git clone git@github.com:ManuelDeAlba/cmd-ext.git

> cd cmd-ext

> npm install

Set a new PATH system variable with the directory path
> C:\cmd-ext\commands

## Help
Commands list
> cmd-ext help

## Update
Get the latest changes
> cmd-ext update

## Use
### ls
> ls [dir]

### cat
> cat [file]

### fetch
> fetch [url] ... [url]

### touch
> touch [name] ... [name]
